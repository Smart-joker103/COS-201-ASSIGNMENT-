# Personal Income Tax Calculator (2009)

This is a Python program that calculates United States personal income tax 
based on the 2009 tax brackets.

## Features
- Supports four filing statuses
- Uses marginal tax rates
- Accepts user input from the console

## Filing Status Codes
- 0: Single
- 1: Married Filing Jointly / Qualifying Widow(er)
- 2: Married Filing Separately
- 3: Head of Household
